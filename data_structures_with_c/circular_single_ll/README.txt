This application is to understand how operations like add, delete and traverse through circular single linked list.

User after running this application, have to give inputs according to the operation that needed to perform.
This application supports the following tasks on circular single linked list.
1. Add a node at the end of list
2. Add a node at the beginning of the list
3. Delete a specific node.
4. Delete a node at the end of the list.
5. Delete a node at the beginning of the list.
6. display all the elements in the list
